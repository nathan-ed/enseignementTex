\contenu{
\tcblower
Soit \(P\) un polynôme non constant
\[
P(x) = a_n x^n + \cdots + a_1 x + a_0.
\]
Montrer que, entre deux racines consécutives de l’équation \(P(x) = 0\), il existe au plus une racine de l’équation \(P'(x) = 0\).
}

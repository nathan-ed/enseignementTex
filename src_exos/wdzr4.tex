\contenu{
\tcblower
Calculer la dérivée d'une fonction constante $f(x)=c$ pour $c\in \mathbb{R}$. 
}
\correction{
\tcblower
Par la définition
\[\lim_{h\to 0} \dfrac{c-c}{h}=\lim_{h\to 0}\dfrac{0}{h}=\lim_{h\to0}0=0.\]
}

\contenu{
\tcblower
Trouver la ou les valeurs prévues par le
théorème des accroissements finis pour la
fonction $f(x) = x^2 - x$ sur
l'intervalle $[-3;4]$
}
\correction{
\tcblower
$f(x) = x^2-x, [-3; 4]$. $f'(c) = \dfrac{12-12}{7} = 0 \implies 2c-1=0 \implies c=\dfrac{1}{2}$.
}

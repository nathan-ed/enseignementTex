\contenu{
\tcblower
Développer et réduire le produit: $\left(n^2+n+1\right)\left(n^2-n+1\right)$. 

$\left(^*\right)$ Déterminer toutes les valeurs de l'entier naturel $n$, pour lesquelles $n^4+n^2+1$ est un nombre premier.
}
\correction{
\tcblower
On développe.

	$\begin{aligned}\left(n^2+n+1\right)\left(n^2-n+1\right)&=n^4-n^3+n^2+n^3-n^2+n+n^2-n+1=n^4+n^2+1
	\end{aligned}$. 

(*) Demander à l'enseignant si intéressé~!
}

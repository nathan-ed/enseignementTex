\contenu{
\tcblower
Calculer, si elles existent, les limites à  gauche et à droite des fonctions suivantes lorsque $x$ tend vers $a$. 
\begin{tasks}(2)
\task $f(x)=\dfrac{3x - 7}{x - 2},\quad a = 2$
\task $f(x)=\dfrac{5x}{(x + 3)^3},\quad a = -3$
\task $f(x)=\dfrac{-3x + 2}{x + 2},\quad a = -2$
\task $f(x)=\dfrac{3x}{|1 - x|},\quad a = 1$
\task $f(x)=\dfrac{-x^2 + 3x + 5}{x^2},\quad a = 0$
\task $f(x)=\dfrac{x^2 + 3x + 5}{x^2 + 2x},\quad a = -2$
\task $f(x)=\lvert 2x - 10\rvert,\quad a = 5$
\task $f(x)=\dfrac{1}{x + 1} - \dfrac{1}{x^2 - 1},\quad a =~-1$
\end{tasks}
}

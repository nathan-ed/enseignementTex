\contenu{
\tcblower
Prouver que la somme de deux entiers impairs quelconques est un nombre pair.
}
\correction{
\tcblower
Soient $a=2m+1$ et $b=2n+1$ deux nombres impairs. 
\[a+b=2m+1+2n+1=2m+2n+2=2(m+n+1)\]
qui est bien un nombre pair.
}

\task $f(x) = \dfrac{x^2}{1+x^2}$, $x \in [-1; 2]$

$f'(x) = \dfrac{2x(1+x^2) - x^2 \cdot 2x}{(1+x^2)^2} = \dfrac{2x}{(1+x^2)^2} = 0 \Rightarrow x = 0$

Posons $u = \sqrt{x}$ : $2u^2 - 3u + 1 = 0 \Rightarrow (2u-1)(u-1) = 0$

$u = \dfrac{1}{2}$ ou $u = 1$ donc $x = \dfrac{1}{4}$ ou $x = 1$

$f''(x) = 2 - \dfrac{3}{2\sqrt{x}}$

$f''\left(\dfrac{1}{4}\right) = 2 - 3 = -1 < 0$ : maximum local

$4 - 2x^2 = 0 \Rightarrow x = \pm\sqrt{2}$

Valeurs : $f(-2) = 0$, $f(-\sqrt{2}) = -2$, $f(\sqrt{2}) = 2$, $f(2) = 0$

Minimum absolu : $x = -\sqrt{2}$, $f = -2$

Maximum absolu : $x = \sqrt{2}$, $f = 2$

\tcblower
Sur une feuille de format A4, ($29,7$ cm par $21$ cm), on effectue la construction suivante~:
\begin{itemize}
\item on trace une diagonale;
\item on trace le segment reliant un des deux autres sommets au milieu du (grand) côté opposé à ce sommet.
\end{itemize}
L'angle entre ces deux segments semble être un angle droit.
L'est-il vraiment~?
}
\correction{
Non, mais presque l'angle est de $90,0028^\circ$
}

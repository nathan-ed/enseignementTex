La somme de deux nombres réels positifs $x$ et $y$ est 20 . Déterminez, s'ils existent, les nombres $x$ et $y$ appartenant à $\mathbb{R}_{+}$et tels que :
\begin{enumerate}
\item $x y$ est maximal ; $x y$ est minimal ;
\item $x^2+y^2$ est maximal ; $x^2+y^2$ est minimal ;
\item $x^2 y^3$ est maximal ; $x^2 y^3$ est minimal.
\end{enumerate}

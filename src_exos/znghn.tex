Nous devons montrer que si la limite d'une fonction existe en un point, alors elle est unique.

\textbf{Théorème :} Si $\lim_{x \to a} f(x)$ existe, alors cette limite est unique.

\textbf{Démonstration par l'absurde :}

Supposons que $f$ admette deux limites différentes en $a$, notées $L_1$ et $L_2$ avec $L_1 \neq L_2$.

Par définition de la limite :
\begin{itemize}
\item $\lim_{x \to a} f(x) = L_1$ signifie : $\forall \varepsilon > 0, \exists \delta_1 > 0$ tel que $0 < |x - a| < \delta_1 \implies |f(x) - L_1| < \varepsilon$
\item $\lim_{x \to a} f(x) = L_2$ signifie : $\forall \varepsilon > 0, \exists \delta_2 > 0$ tel que $0 < |x - a| < \delta_2 \implies |f(x) - L_2| < \varepsilon$
\end{itemize}

Choisissons $\varepsilon = \dfrac{|L_1 - L_2|}{3} > 0$ (puisque $L_1 \neq L_2$).

Pour ce $\varepsilon$, il existe $\delta_1 > 0$ et $\delta_2 > 0$ comme ci-dessus.

Posons $\delta = \min(\delta_1, \delta_2) > 0$.

Pour tout $x$ tel que $0 < |x - a| < \delta$, nous avons simultanément :
\begin{align*}
|f(x) - L_1| &< \varepsilon = \frac{|L_1 - L_2|}{3} \\
|f(x) - L_2| &< \varepsilon = \frac{|L_1 - L_2|}{3}
\end{align*}

Par l'inégalité triangulaire :
\[|L_1 - L_2| = |L_1 - f(x) + f(x) - L_2| \leq |L_1 - f(x)| + |f(x) - L_2| = |f(x) - L_1| + |f(x) - L_2|\]

Donc :
\[|L_1 - L_2| < \frac{|L_1 - L_2|}{3} + \frac{|L_1 - L_2|}{3} = \frac{2|L_1 - L_2|}{3}\]

Ceci implique :
\[|L_1 - L_2| < \frac{2|L_1 - L_2|}{3}\]

En multipliant par 3 :
\[3|L_1 - L_2| < 2|L_1 - L_2|\]

Donc :
\[|L_1 - L_2| < 0\]

Ce qui est \textbf{absurde} car une distance ne peut pas être négative.

\textbf{Conclusion :}

Notre hypothèse de départ ($L_1 \neq L_2$) mène à une contradiction. Donc si la limite existe, elle est nécessairement unique : $L_1 = L_2$.

\[\boxed{\text{La limite, si elle existe, est unique.}}\]

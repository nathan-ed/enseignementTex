\[C \cup D=\{1 ; 2 ; 3 ; 4 ; 5\}, C \cap D=\{2 ; 3 ; 4\}, 1 \notin D \backslash C \text{ et } 5 \notin C \backslash D\]

\[E \cup F=\{2 ; 3 ; 4 ; 5\} \text{ et } E \cap F=\{2 ; 4\}\] 

 Donner toutes les possibilités.
